Sinatra-MVC
===========

Sinatra project running MVC architecture

##Usage

First install the dependencies: `bundle`

Then from there whenever you want to run the Sinatra app: `bundle exec shotgun`

##Routing

The routing is handled via the `config.ru` file

##Directory Structure

- assets
    - scripts
    - styles
- controllers
- helpers
- models
- views
